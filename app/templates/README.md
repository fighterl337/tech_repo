# Library Recommender System

Веб-приложение на FastAPI для управления библиотекой с системой рекомендаций книг на основе денормализованной базы данных.

## Реализованный функционал

- Каталог книг (более 1000 реальных произведений)
- CRUD книги, читатели, выдача/возврат
- Рекомендации "С этой книгой также берут" на основе предрассчитанной таблицы `book_co_occurrence`
- API `/recommendations/for-book/{book_id}` для получения рекомендаций
- Денормализованное хранение пар книг с весами (быстрые запросы без JOIN)
- Логирование действий пользователей (`user_activities`)

## Технологический стек

- Python 3.11+
- FastAPI
- SQLAlchemy + SQLite
- Jinja2
- Faker

## Быстрый запуск

Установить зависимости:

```bash
pip install -r requirements.txt
```
Заполнить базу данных книгами и построить денормализованную таблицу рекомендаций: 

```bash
python clean_books.py
python fill_co_occurrence.py
```
Запустить сервер:

```bash
uvicorn app:app --reload
```
Открыть в браузере:

```bash
http://127.0.0.1:8000/books_page
```
## Демонстрация
На странице каждой книги отображается блок "С этой книгой также рекомендуют". Рекомендации основаны на предрасчитанных связях между книгами (одни авторы, близкие эпохи) – пример денормализации для быстрых запросов.
